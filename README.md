## 1. Verdure EMO

一个为OTTO小智机器人改进的外壳，主要包含头部的修饰，将头部转换成一个EMO机器人的造型。

注意事项和额外配件物料购买链接请到**verdure-emo**目录查看。

渲染图如下：

[![B站视频演示链接](/verdure-emo/images/otto-emo.jpg)](https://www.bilibili.com/video/BV1gpL6z9E5N/)
